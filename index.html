generator-nomatic-web-material

Yeoman generator for a Vue.js project using Material Design UI components

Installation

First, install Yeoman and generator-nomatic-web-material using npm (we assume you have pre-installed node.js).

npm install -g yo
npm install -g generator-nomatic-web-material

Then generate your new project:

yo nomatic-web-material
